# В этом примере приведены примеры простейшей реализации алгоритмов сокращения частоты вызова обработчиков слишком большом количестве событий


## Как пользоваться

```
    npm install
    npm run start
```

Также необходимо запустить локальный REST сервер:

```
node ./server.mjs

```
