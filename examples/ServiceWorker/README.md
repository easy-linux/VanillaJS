# Web service example

## **Добавляем возможность офф-лайновой работы для любого сайта или веб приложения**


## Как пользоваться

```
    npm install
    npm run build
```
или

```
   npm install
   npm run start
```

или

```
   npm run build
   cd build
   npx http-server-spa ./ ./index.html 5001
```
